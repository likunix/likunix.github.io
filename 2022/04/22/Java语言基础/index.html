    变量与数据类型

    命名规范

    • 类名

      • 驼峰命名法，大写字母开头，由多个单词组成的，所有单词首字母均要大写

      • 接口类以大写字母I开头，接口的实现类去掉首字母I，并以ImpI作为结尾

    • 方法名

      驼峰命名法，但是首字母要小写

    • 变量

      • 普通变量用驼峰命名法，但是首字母要小写，可在变量名前加上2到3个字母的前缀表示变量的数据类型
      • 静态变量全部字母大写，单词间常用_连接
      • 常量全部字母大写，单词间常用_连接，且要用有意义的单词定义

    数据类型

    Java中有8中基本数据类型

    分类 数据类型
    6种数字类型(四整型二浮点) byte,short,int,long,float,double
    1种字符类型 char
    1种布尔型 boolean

    • byte

      8位，有符号，以二进制补码表示的整数

      默认值0，最小值-128(-2^7) ，最大值127(2^7-1)

    • short

      16位，有符号，以二进制补码表示的整数

      默认值0，最小值-32,768(-2^15), 最大值32,768(2^15-1)

    • int

      32位，有符号，以二进制补码表示的整数

      默认值0，最小值-2,147,483,648(-2^31), 最大值2,147,483,648(2^31-1)

    • long

      64位，有符号，以二进制补码表示的整数，Java中使用要在值后加L，否则作整型解析

      默认值0L，最小值是 -9,223,372,036,854,775,808（-2^63）

      最大值是 9,223,372,036,854,775,807（2^63 -1）

    • float

      32位，单精度，符合IEEE 754标准的浮点数

      默认值0.0f

      浮点数不能用来表示精确的值，如货币(用BigDecimal)

    • double

      64位，双精度，符合IEEE 754标准的浮点数

      默认值0.0d

    • char

      表示单一的 16 位 Unicode 字符

      单引号赋值

      最小值\u0000(0), 最大值\uffff(65535)

    • boolean

      表示一位的信息

      只有两个取值：true 和 false

      默认值false

    此外，每种基本数据类型对应一种包装类

    基本类型 包装类
    boolean Boolean
    char Character
    int Integer
    byte Byte
    short Short
    long Long
    float Float
    double Double

    运算符与表达式

    算术运算符

    操作符 描述
    + 加法 - 相加运算符两侧的值
    - 减法 - 左操作数减去右操作数
    * 乘法 - 相乘操作符两侧的值
    / 除法 - 左操作数除以右操作数
    % 取余 - 左操作数除以右操作数的余数
    ++ 自增: 操作数的值增加1
    – 自减: 操作数的值减少1

    关系运算符

    运算符 描述
    == 两个操作数的值相等，条件为真
    != 两个操作数的值不相等，条件为真
    > 左操作数的值大于右操作数的值，条件为真
    < 左操作数的值小于右操作数的值，条件为真
    >= 左操作数的值是大于或等于右操作数的值，条件为真
    <= 左操作数的值小于或等于右操作数的值，条件为真

    位运算符

    操作符 描述
    & 两个操作数对应位都是1，则结果为1，否则为0
    | 两个操作数对应位都是0，则结果为0，否则为1
    ^ 两个操作数对应位值相同，则结果为0，否则为1
    ~ 将操作数按位取反
    << 按位左移运算符
    >> 按位右移运算符
    >>> 按位右移补零操作符，移动得到的空位以零填充

    逻辑运算符

    操作符 描述
    && 逻辑与运算符
    || 逻辑或操作符
    ! 逻辑非运算符
