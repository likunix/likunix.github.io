    标题

    # ... 一级标题

## ... 二级标题

### ... 三级标题

#### ... 四级标题

##### ... 五级标题

###### ... 六级标题

    注意:

    • #...标记后要接空格再加标题内容
    • 为方便使用书签跳转阅读，要点使用五级标题标记
    • 本文基于Typora编辑器，不同编辑器的功能(如快捷键)可能会有一些差别

    HTML

    注释

    • <!--...-->浏览器不显示注释，下面空格前的一行注释在浏览器中不显示

    空格

    • markdown中空格大部分时候都是起着排版控制作用，因此没法在行首或格式控制符之后输入空格

      此时，可直接输入HTML entities中的空格占位符

    超链接

    • <a>...</a> 锚点

      <a> 标签用来实现跳转到指定锚点(anchor)或超链接（hyperlink）

      <a> 元素最重要的属性是 href 属性，它指示链接的目标，可以是页内锚点，也可以是另一个站点页面

    • <img />链接一幅图像

      <img> 标签创建的是被引用图像的占位空间

      <img> 标签有两个必需的属性：src 属性 和 alt 属性

      属性 值 描述
      Alt text 图像的替代文本
      src URL 图像的URL

    标题

    • <h1> - <h6> <h1> 定义最大的标题，<h6> 定义最小的标题

    文字格式

    • <em>...</em> 强调文本

      <i>...</i> 斜体

    • <strong>...</strong>重要文本（粗体）

      <b>...</b> 粗体

    • <mark>...</mark> 突出显示文本（高亮）

    • <big>...</big> 加大字号

      <small>...</small> 减小字号

    • <sup>...</sup> 上角标

      <sub>...</sub> 下角标

    • <u>...</u> 下划线

    • <del>...</del> 删除线

    排版控制

    • <hr>分隔水平线，无需闭合

    • <br> 插入换行符


    • <p>...</p> 定义段落，前后自动创建一些空白

      这是一个段落

    • <div>...</div>把文档分割为独立的、不同的部分,用 id 或 class 来标记 <div>，可更有效地使用(文字着色)

    • align属性align 属性规定元素（<h1> - <h6>、<div>）的水平对齐方式

      HTML 4.01 中，不推荐使用 <h1> - <h6> 元素的 align 属性；

      在 XHTML 1.0 Strict DTD 中，不支持 <h1> - <h6> 元素的 align 属性。用 CSS 代替:<h1 style="text-align:right">

    引用

    • <cite>...</cite>表示它所包含的文本对某个参考文献的引用，如标题

      按照惯例，引用的文本将以斜体显示

    • <q>...</q>短引用(行内引用)

      q 元素中的 cite 属性指定了引用的来源，但主流浏览器均不支持 cite 属性，可以当做注释用

    • <blockquote>...</blockquote>定义摘自另一个源的块引用

      标签之间的所有文本都会从常规文本中分离出来，经常会在左、右两边进行缩进，而且有时会使用斜体

      q 元素中的 cite 属性同上

    • <code>...</code> <code> 标签通常只是把文本变成等宽字体，但它暗示着这段文本是源程序代码

      如果只是希望使用等宽字体的效果，请使用 <tt> 标签

      如果想要在严格限制为等宽字体格式的文本中显示编程代码，请使用 <pre> 标签

    • <pre>...</pre>定义预格式化的文本，被包围在 pre 元素中的文本通常会保留空格和换行符，而文本也会呈现为等宽字体

      一个常见应用就是用来表示计算机的源代码

    清单

    • - [ ] sth ，注意中括号及前后的空格，sth为清单内容，[ ]项完成后为[x]。

      因为-space 会自动识别为列表项，故可编辑完成后再粘贴到Typora

    表格

    • Markdown 使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行

        |  表头   | 表头  |
  |  ----  | ----  |
  | 单元格  | 单元格 |
  | 单元格  | 单元格 |

    • 对齐方式

      • :- 设置内容和标题栏居左对齐
      • :-: 设置内容和标题栏居中对齐
      • -: 设置内容和标题栏居右对齐
