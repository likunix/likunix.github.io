    基本概念

    面向对象三特征

    • 封装性

      有两个含义

      一是把对象的属性和行为看成一个密不可分的整体,将两者”封装”在独立单位(对象)中

      二是”信息隐蔽”,把不需要外界知道的信息隐藏起来,并尽可能屏蔽对象的功能实现细节

    • 继承性

      在已有一般类的基础上，派生出其他反映特殊事物的类

      被继承的类称为父类或超类，通过继承产生的类称为子类或派生类

    • 多态性

      多态指同一个实体同时具有多种形式

      方法重载就是一种多态，同一个类中允许多个方法重名，调用时根据方法签名来具体确定调用的方法

    类与对象

    类表示一个客观世界中某类群体的一些基本特征抽象，对象表示一个个具体的东西

    类是由属性和方法组成，属性中定义类需要的具体信息，用变量表示；方法是一些操作的行为

    对象使用前必须实例化

    封装性

    封装性是面向对象的第一大特性，所谓封装性也就是指对外部不可见

    例如对于类中的属性，类型应当设置为private，这些封装属性，应当提供setter和getter方法来设置和取得

    构造方法

    构造方法无返回值，且名字要与类名一致

    每个类至少有一个构造方法，若不定义，则自动生成一个无参、什么都不做的构造方法，定义了，则不生成，此外，与普通方法一样，构造方法可以被重载

    匿名对象

    匿名对象指不声明直接实例化出来的对象，匿名对象只在堆内存中开辟空间，不存在栈内存的引用，使用一次后等待GC回收，一般用于为其他实例化对象的参数传递
